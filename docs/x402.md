# x402 Payment Protocol — qsearch Integration

## Overview

qsearch v0.3 ships a **payment middleware skeleton** at `src/x402/middleware.js` implementing the [x402 protocol](https://x402.org) — an HTTP 402 "Payment Required" standard for machine-to-machine micropayments over blockchain networks.

## Status

- **v0.3 (current):** `X402_ENABLED=false` — middleware is installed but passthrough. All requests succeed unconditionally.
- **v0.4 (planned):** `X402_ENABLED=true` — on-chain signature verification via x402 SDK. Agents that don't attach a valid payment header receive HTTP 402.

## Protocol Flow (v0.4)

```
Agent → POST /search (no payment)
Server → 402 Payment Required
         X-PAYMENT-REQUIRED: <base64 JSON>
         {
           "network": "base",
           "asset": "USDT",
           "amount": "0.01",
           "recipient": "0x...",
           "description": "qsearch API access"
         }

Agent → (signs micropayment on Base network)
Agent → POST /search
         X-PAYMENT-SIGNATURE: <signature>
Server → 200 OK (results)
```

## Configuration

| Variable | Default | Description |
|---|---|---|
| `X402_ENABLED` | `false` | Set `true` to enforce payment |
| `X402_NETWORK` | `base` | Blockchain network (Base L2) |
| `X402_PRICE_USDT` | `0.01` | Price per request in USDT |
| `X402_RECIPIENT` | — | Wallet address to receive payment |
| `X402_FACILITATOR_URL` | — | x402 facilitator endpoint for verification |

## Why x402 + Base

- **Base** is Coinbase's L2 — low fees (~$0.001/tx), fast finality, USDT native support
- **x402** is an emerging standard for autonomous agent payments — no OAuth, no API keys, just on-chain proof
- **Use case:** AI agents that self-fund API access without human intervention

## Implementation Notes

The `requirePayment(handler, opts)` wrapper:
- `opts.skipIf(req)` — allowlist function (e.g. skip `/health`, CORS preflight, dev IPs)
- `opts.priceUsdt` — override per-endpoint price
- `opts.recipient` — override per-endpoint recipient

To enable in v0.4:
```bash
X402_ENABLED=true
X402_RECIPIENT=0xYourWalletAddress
X402_PRICE_USDT=0.001
```

Then wrap any handler:
```javascript
import { requirePayment } from './x402/middleware.js'

// In HTTP server routing:
if (req.url === '/search') {
  requirePayment(handleSearch, {
    skipIf: (r) => r.headers['x-dev-bypass'] === process.env.DEV_SECRET
  })(req, res)
}
```
